# Платформа управления клиниками

Полнофункциональное веб-приложение для управления взаимодействием продавцов с клиниками.

## Требования

- Python 3.9+
- Flask 2.3+
- SQLite

## Установка

```bash
# Создать виртуальное окружение
python3 -m venv venv

# Активировать
source venv/bin/activate  # Linux/Mac
venv\\Scripts\\activate     # Windows

# Установить зависимости
pip install -r requirements.txt

# Создать первого администратора
python create_admin.py

# Запустить приложение
flask run
```

Приложение будет доступно по адресу: http://localhost:5000

## Функциональность

### Для администраторов:
- ✅ Управление пользователями (создание, редактирование, удаление)
- ✅ Управление продуктами и их ценами
- ✅ Назначение продуктов продавцам
- ✅ Рассмотрение и одобрение заявок на авторизацию
- ✅ Загрузка общих материалов и персональных документов
- ✅ Управление курсом валюты
- ✅ Система календаря с подтверждением запросов
- ✅ Настройка SMTP и отправка email
- ✅ Отслеживание всех заявок и событий

### Для продавцов:
- ✅ Просмотр назначенных продуктов с расчётом цен со скидкой
- ✅ Подача заявок на авторизацию клиник
- ✅ Отслеживание статуса заявок (pending → approved → expired)
- ✅ Повторная авторизация истекших клиник
- ✅ Создание запросов в календаре
- ✅ Просмотр анонимного календаря с занятыми слотами
- ✅ Скачивание общих материалов
- ✅ Просмотр персональных документов

## Развёртывание

### Development

```bash
export FLASK_ENV=development
flask run
```

### Production (с gunicorn)

```bash
pip install gunicorn
gunicorn --workers 4 --bind 0.0.0.0:8000 app:app
```
